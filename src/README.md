# 📝 React Todo App

A simple but powerful Todo App built with React, focused on understanding state management, component structure, and real-world UI behavior.

---

## 🚀 Live Demo
https://react-todo-app-xi-rouge.vercel.app

---

## ✨ Features

- Add new tasks  
- Delete tasks  
- Edit tasks  
- Mark tasks as completed  
- Select All / Unselect All  
- Search functionality  
- Persistent storage using localStorage  
- Responsive design (mobile-friendly)

---

## 🧠 What I Learned

This project helped me understand:

- React state management with useState  
- Component structure and prop handling  
- useEffect for side effects (localStorage)  
- Array methods like map(), filter(), every()  
- Handling forms and events in React  
- Debugging real issues (small mistakes affecting logic)  
- Improving UI/UX for better user experience
